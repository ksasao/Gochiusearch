■ 概要
ご注文はうさぎですか？1期・2期・Dear My Sister・Sing for You・3期 の画像から何羽のどのあたりかを
推測するアプリです。

■ 動作条件
- Windows 10 / .NET Framework 4.7.1
- Mac OS X 10.11
- Microsoft Edge 25以降, Firefox 43以降, Chrome 48以降, Opera 35以降
  (IE11 または Macではブラウザから直接画像がDrag&Dropできません)
- Mono 4.2以降(Mac版)
  http://www.mono-project.com/download/#download-mac から.pkgを
  ダウンロードしてください。

■ インストール・アンインストール
(1) ZIP ファイルを利用した場合
インストーラーはありません。
アンインストールする場合はフォルダごと削除してください。

(2) Windowsストアを利用した場合
https://www.microsoft.com/ja-jp/p/%E3%81%94%E3%81%A1%E3%81%86%E3%82%B5%E3%83%BC%E3%83%81/9njb7xgz6tk5
アプリと機能から削除できます

■ 利用方法
Gochiusearch.exe を直接起動してください。
市松模様部分にローカルの画像もしくはブラウザに表示された
画像をDrag&Dropするとその画像から羽数などを推定します。


(注)
- ブラウザに表示された画像はブラウザの仕様やコンテンツ配信
  元の設定によりDrag&Dropできないことがあります
- ニコニコ動画の当該動画の閲覧は第1羽を除き有料です
  (2021/1/6現在)
- ニコニコ動画で動画を自動再生するには、プレミアム会員となり
  動画の右下の歯車アイコン>動画再生>(ページを開いたときに)
  動画を自動再生 にチェックを入れてください。
